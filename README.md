A comprehensive data analysis project examining customer shopping patterns, purchasing behavior, and business trends using SQL, Python, and Power BI.

This project analyzes customer shopping behavior data to uncover insights about purchasing patterns, revenue generation, customer segmentation, and product performance. The analysis combines data processing with Python, SQL database queries, and interactive Power BI dashboards.

- Identify high-value customers and segment them by behavior

## üìä Dataset Overview

**Source File:** `customer_shopping_behavior.csv`

**Records:** 500+ customer transactions

**Key Attributes:**
- **Customer Info:** Customer ID, Age, Gender, Location
- **Purchase Details:** Item Purchased, Category, Purchase Amount (USD)
- **Product Attributes:** Size, Color, Season
- **Behavioral Data:** Review Rating, Subscription Status, Previous Purchases
- **Transaction Details:** Shipping Type, Discount Applied, Payment Method, Frequency of Purchases

**Categories Analyzed:**
- Clothing, Footwear, Accessories, Outerwear

## üìà Key Analysis Questions

The project addresses 10 core business questions:

1. **Revenue Analysis:** Total revenue generated by male vs. female customers
2. **Discount Impact:** Which discounted customers spent above average?
3. **Product Ratings:** Top 5 products by average review rating
4. **Shipping Comparison:** Average purchase amounts by shipping type
5. **Subscription Value:** Do subscribers spend more than non-subscribers?
6. **Discount Trends:** Top 5 products with highest discount rates
7. **Customer Segmentation:** Classification into New, Returning, and Loyal customers
8. **Category Leaders:** Top 3 products within each category
9. **Repeat Buyers:** Subscription behavior of repeat customers (5+ purchases)
10. **Age Demographics:** Revenue contribution by age group

## üöÄ Getting Started

### Prerequisites
- Python 3.7+
- PostgreSQL or MySQL database
- Power BI Desktop (for dashboard visualization)
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd customer-trends-data-analysis-SQL-Python-PowerBI
   ```

2. **Set up Python environment**
   ```bash
   python -m venv .venv
   # On Windows:
   .venv\Scripts\activate
   # On macOS/Linux:
   source .venv/bin/activate
   ```

3. **Install required packages**
   ```bash
   pip install pandas sqlalchemy psycopg2-binary python-dotenv
   ```

4. **Configure database connection**
   - Create a `.env` file in the project root
   - Add your database credentials:
     ```
     DATABASE_URL=postgresql://username:password@localhost:5432/customer_db
     ```

5. **Run Python analysis**
   - Open `Customer_Shopping_Behavior_Analysis.ipynb` in Jupyter Notebook
   - Execute cells sequentially to process data and load into database

### Running SQL Queries

Execute queries from `customer_behavior_sql_queries.sql` in your database client:
```sql
-- Example: Get total revenue by gender
SELECT gender, SUM(purchase_amount) as revenue
FROM customer
GROUP BY gender;
```

### Opening Power BI Dashboard

1. Open `customer_behavior_dashboard.pbix` in Power BI Desktop
2. Configure data source connections if needed
3. Interact with visualizations and filters

## üîç Data Processing Pipeline

### Python Processing (`Customer_Shopping_Behavior_Analysis.ipynb`)

1. **Data Loading:** Import CSV file using Pandas
2. **Data Exploration:** Check structure, info, and statistics
3. **Missing Values:** Impute Review Rating with category median
4. **Column Normalization:** Convert to snake_case for consistency
5. **Feature Engineering:**
   - Age groups: Young Adult, Adult, Middle-aged, Senior
   - Purchase frequency in days for trend analysis
6. **Database Connection:** Load cleaned data to PostgreSQL/MySQL
7. **Data Validation:** Verify successful data transfer

### SQL Analysis

Structured queries cover:
- Aggregation and grouping
- Subqueries and CTEs (Common Table Expressions)
- Window functions (ROW_NUMBER)
- Case statements for segmentation
- Joins and filtering
